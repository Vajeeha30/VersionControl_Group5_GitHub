Title: GitHub — Version Control Platform
Introduction 
GitHub is a widely used online platform that supports version control and team-based software development. It is built on the Git system, which helps developers record changes, maintain project history, and work on code collaboratively.By using GitHub, several developers can contribute to the same project at the same time without overwriting each other’s updates.
